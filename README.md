# Capterra

Capterra (https://www.capterra.com/) is a B2B software review and comparison platform owned by Gartner. It helps business buyers discover, evaluate, and select software through verified user reviews, side-by-side product comparisons, pricing information, and curated shortlists across hundreds of software categories. Capterra is part of the Gartner Digital Markets network, which also includes GetApp and Software Advice.

## Overview

Founded in 1999 and acquired by Gartner in 2015, Capterra hosts millions of verified reviews across thousands of software products. Buyers can filter software by features, pricing model, deployment type, and company size. Vendors can list their products, manage their profiles, collect reviews, and participate in Capterra's pay-per-click (PPC) advertising program to improve visibility in category listings. The platform awards "Best Of" badges annually to top-rated products based on review volume and ratings.

## Key Features

- Verified user reviews for thousands of B2B software products
- Side-by-side software comparison tools
- Category-based browsing across hundreds of software verticals
- PPC advertising program for software vendors
- Best Of badges recognizing top-rated products per category
- Vendor portal for managing listings, reviews, and ad campaigns
- Integration with Gartner Digital Markets (GetApp, Software Advice)

## APIs

Capterra provides a vendor-facing Click Report API that allows software vendors to programmatically retrieve historical PPC click data. Access requires an API key obtained through a Capterra account manager. API documentation is available within the vendor portal under PPC Bidding settings after login.
